# Face Detection Project

This project performs real-time face detection using Python and OpenCV.

## Features
- Detects faces from webcam video feed.
- Highlights detected faces with bounding boxes.
- Easy to run and extend.

## Requirements
- Python 3.x
- OpenCV library
